# BodyParts3D Asset Folder

This folder is the local target for anatomy meshes used by the 3D body cards.

Current content:
- `base_humanoid.obj`: demo mesh to validate the full 3D rendering pipeline.

For higher realism:
1. Export or convert BodyParts3D anatomical meshes to OBJ.
2. Place refined meshes in this folder.
3. Update layer mapping in `src/core/anatomy_assets.py`.
